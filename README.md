# 🚁 Real-Time Human Detection & Counting using YOLOv8

This project implements a real-time human detection and counting system using YOLOv8 (Ultralytics).  
It supports both:

- 🎥 Webcam-based live human detection
- 🖼 Image-based human counting

The system detects people in a frame and displays the total number of humans detected.

---

## 📌 Features

- Real-time human detection using YOLOv8
- Frame-wise human counting
- Bounding box visualization
- Image-based human counting
- Clean and modular Python implementation
- Virtual environment support

---

## 🧠 Technologies Used

- Python 3.11
- OpenCV
- Ultralytics YOLOv8
- PyTorch
- NumPy
- Pillow (PIL)
